# TheGuild3 — Українська локалізація

Короткий опис
-
Проект призначений для створення та підтримки української локалізації гри The Guild 3. Ресурс містить оригінальні файли локалізацій, робочі файли перекладу та готові файли модифікації для розміщення в Steam Workshop.

Статус
-
Робота в процесі — файли перекладу частково готові. Далі заплановано перевірку в грі, виправлення термінології та публікацію модифікації.

Структура репозиторію (головні директорії)
-
- **`source/origin`**: оригінальні файли локалізацій (джерело для перекладу).
- **`translation/mod/G3Ua_Steam`**: підготовлений мод для публікації (скрипти `publish_mod.bat`, `update_mod.bat`, `steam_appid.txt`, і структура `content/media/localization`).
- **`translation/mod/mod_source/localization`**: робочі файли локалізації, які використовуються під час перекладу.
- **`translation/mod/mod_source/POEdit`**: файли `.po`/`.pot`/`.looo` для роботи в POEdit та для збереження перекладів.

Як тестувати локалізацію локально
-
1. Переконайтесь, що останні переклади знаходяться в `translation/mod/mod_source/localization`.
2. Скопіюйте або автоматично оновіть файли в пакеті мода: перейдіть у `translation\\mod\\G3Ua_Steam` і запустіть `update_mod.bat` (PowerShell/командний рядок):

```powershell
cd translation\\mod\\G3Ua_Steam
.\\update_mod.bat
```

3. Після цього запустіть гру із директорії мода або помістіть `content` у потрібну папку гри, аби перевірити переклад у грі.

Публікація у Steam Workshop
-
Використовуйте `publish_mod.bat` в `translation/mod/G3Ua_Steam` для підготовки та завантаження модифікації у Steam Workshop. Переконайтесь, що `steam_appid.txt` містить правильний ID і що ви авторизовані в SteamCMD/Steam.

Внесок
-
- Для редагування перекладів використовуйте PO-файли з `translation/mod/mod_source/POEdit` (наприклад `locdirect_ukrainian.po`).
- Створюйте pull request або відкривайте issue з описом змін.
- Дотримуйтесь консистентності термінології — див. `docs/translation_guide.md.txt`.

Рекомендації по ліцензії
-
Додайте файл `LICENSE`, щоб вказати умови використання перекладів. Для перекладів часто використовують Creative Commons або MIT/ISC для вихідних файлів; узгодьте вибір з авторами модифікацій або відповідно до політик гри.
